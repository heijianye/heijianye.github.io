## 项目概述 | Project Overview

- 项目名称：黑眼圈的个人主页
- 响应式设计：支持移动端和桌面端 # 采用移动优先设计理念
- 设计风格：现代简约，紫色主题 # 参考当代设计趋势

## 功能特点 | Features

- 响应式导航栏，支持移动端菜单 # 自适应不同设备尺寸
- 平滑滚动效果 # 使用JavaScript实现页面平滑过渡
- 卡片式布局展示技能和工具 # 采用现代卡片UI设计
